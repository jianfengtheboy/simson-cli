# simson-cli
sim构建CLI，提供项目模版快速创建移动端、管理系统等项目

### 安装方法
```shell
  npm install simson-cli -g

  // 更新
  npm update simson-cli -g
```

### 使用方法
```shell
  // 创建项目
  sim create
  // 模版管理
  sim template
```

### 项目

#### sim create，创建项目
根据提示选择要创建的项目即可

### 模版

#### sim template，模版管理

* -ls 列出所有模版

```
  sim template -ls
```

* -add 增加模版

```
  sim template -add
```
